# 错误码

本指南包括您可能从 API 中看到的错误码以及描述信息，描述信息中包含了每个错误码的原因以及进一步的解决方案。

## 常见错误码

| 错误码 | 原因                                                                                                                | 解决方案                     |
| :-: | ----------------------------------------------------------------------------------------------------------------- | ------------------------ |
| 400 | 请求参数格式不正确，可能原因如下：  <br />  1.图片无法下载 <br />  2.图片数量超过限制 <br />  3.该模型不支持视频输入  <br />  4.模型不存在或无权限 <br />  5.参数值不合法 | 请参照文档正确传递参数信息            |
| 401 | 认证无效                                                                                                              | 确保使用正确的 API 密钥           |
| 402 | 余额不足                                                                                                              | 确保账户里面有足够余额              |
| 404 | 请求路径不正确                                                                                                           | 请参照文档修复请求路径信息            |
| 429 | 请求的资源超限，可能原因是你发送请求太快，超过了速率限                                                                                       | 请稍候重试您的请求                |
| 451 | 请求内容或者响应内容未审核通过                                                                                                   | 修改请求信息后再重试               |
| 500 | 我们服务器上的问题                                                                                                         | 稍等片刻后重试请求，如果问题仍然存在，请联系我们 |
| 503 | 目前服务器负载过高                                                                                                         | 请稍候重试您的请求                |
